# Web UI 使用指南

## 快速启动

```bash
cd web
python app.py
```

访问 **http://localhost:5000** 即可使用！

## 配置 LLM 后端

1. 访问 Web UI，点击 **系统设置**

2. 填写配置：
   - **后端地址**：如 `http://127.0.0.1:11434/`
   - **API Key**：本地服务留空，云服务填写密钥
   - **模型名称**：本地服务留空，云服务填写模型名

3. 点击 **测试连接** 验证配置

4. 点击 **保存配置** 自动写入配置文件

### 常用后端配置示例

| 后端 | 后端地址 | API Key | 模型名称 |
|------|----------|---------|----------|
| llama.cpp | `http://127.0.0.1:8080/` | 留空 | 留空 |
| DeepSeek | `https://api.deepseek.com/` | 您的密钥 | `deepseek-chat` |
| 通义千问 | `https://dashscope.aliyuncs.com/compatible-mode/v1/` | 您的密钥 | `qwen-plus` |
| OpenAI | `https://api.openai.com/v1/` | 您的密钥 | `gpt-3.5-turbo` |

## 使用流程

### 1. 上传文档

- 拖拽 `.txt` 或 `.docx` 文件到上传区域
- 或点击选择文件
- 支持最大 500MB

### 2. 选择处理模式

| 模式 | 适用场景 | 输出格式 |
|------|----------|----------|
| **全文模式** | 生成论文总结与展望 | `.txt` |
| **章节模式** | 逐章生成总结 | `.txt` |
| **段落模式** | 语法检查与润色 | `.txt` + `.docx` |

### 3. 配置参数

- **提示词版本**：选择不同的提示词版本（段落模式支持多版本切换）
- **其他配置**：根据模式不同会有相应选项

### 4. 开始处理

- 点击 **开始处理** 按钮
- 自动跳转到任务进度页面
- 实时显示处理进度和日志

### 5. 查看结果

- 处理完成后可以 **预览** 结果文件
- 点击 **下载** 保存到本地
- 支持查看历史任务记录

## 🎨 Web UI 功能特性

### 核心功能
- 📤 **拖拽上传**：支持 `.txt` 和 `.docx` 格式
- ⚙️ **可视化配置**：模式选择、提示词版本配置
- 📊 **实时进度**：WebSocket 推送处理进度和日志
- 📝 **任务管理**：查看历史任务、下载结果文件
- 🔧 **系统设置**：LLM 后端配置和连接测试
- 🎨 **现代界面**：响应式设计、中文本地化

### 提示词管理

Web UI 支持完整的提示词版本管理：

1. **查看版本**：在提示词配置页面查看所有可用版本
2. **切换版本**：处理任务时选择不同的提示词版本
3. **创建版本**：自定义新的提示词版本
4. **编辑版本**：修改已有的自定义版本
5. **删除版本**：删除不需要的自定义版本

> ⚠️ **注意**：默认的 `default` 版本不可修改或删除，以保留原有配置。

## ⚠️ 注意事项

1. **确保 LLM 服务已启动**
2. **文件大小**：最大支持 500MB
3. **浏览器兼容**：推荐使用 Chrome/Edge 等现代浏览器
4. **网络连接**：任务页面使用 CDN 加载 Socket.IO 客户端

## 🐛 故障排除

### 端口占用
```bash
# 检查端口占用
netstat -ano | findstr :5000

# 修改端口：编辑 web/app.py 最后一行
socketio.run(app, host='0.0.0.0', port=8080, debug=True)
```

### 依赖缺失
```bash
pip install flask flask-socketio simple-websocket
```

### Socket.IO 连接失败
- 检查防火墙设置
- 确认任务页面加载了 Socket.IO CDN
- 查看浏览器控制台错误信息
